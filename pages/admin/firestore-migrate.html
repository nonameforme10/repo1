RTDB to Firestore Migration

This copies the old Realtime Database data for phones, global_chat, and online, weekly challenges, plus readings, listening, and vocabularies into Firestore so the new app code can read the same data from Firestore.

Phones Migrates the phone index so phone-based sign in keeps working.
Global Chat Copies room metadata, messages, and live typing docs into Firestore.
Online Lessons Moves lesson modules and lesson comments into Firestore collections.
Readings Copies reading tests and pass answer keys into Firestore part documents.
Listening Copies listening tests and section answer keys into Firestore section documents.
Vocabularies Moves unit words, listening words, and password hashes into Firestore, and copies live waiting rooms into RTDB vocabularyRooms.
Admins Seeds your own Firestore admin doc so Firestore rules can recognize you.
